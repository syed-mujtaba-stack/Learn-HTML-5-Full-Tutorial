Progressive Web Apps (PWA) Essentials - Exercise

1. Web App Manifest

Create a manifest.json with name, short_name, icons, and theme_color. Link it in the HTML <head>.

Hint: Use "start_url", "display", and at least one icon (192x192 or 512x512).
  • File: manifest.json
  • Link: <link rel="manifest" href="manifest.json">

2. Service Worker Registration

Register a service worker (sw.js) that caches site assets for offline use.

Hint: Use navigator.serviceWorker.register() and cache index.html, CSS, and JS files in the install event.

3. Add to Home Screen

Test PWA installability using Chrome or Edge. Make sure the manifest and service worker are working.

Hint: You should see an install prompt or "Add to Home Screen" option in the browser menu.

4. Testing PWA Features

  • Test offline mode by disconnecting from the network.
  • Check the "Application" tab in DevTools for manifest and service worker status.
  • Try installing the app on your device.
